# Microservice Deployment using VirtualBox

This repository contains a simple Node.js microservice deployed on a Puppy Linux
virtual machine and accessed from another VM to demonstrate inter-VM communication.

## Technologies Used
- Node.js
- Express.js
- Oracle VirtualBox
- Puppy Linux & Alpine Linux

## How to Run
1. Install Node.js and npm
2. Run `npm install`
3. Start the service using `npm start`
4. Access the service via `http://<VM-IP>:3000`
